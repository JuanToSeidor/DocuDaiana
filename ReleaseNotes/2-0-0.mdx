## ✨ Novedades

- **Nueva interfaz de usuario**: Rediseñamos la experiencia para hacerla más intuitiva y accesible.
- **Compatibilidad con nuevas fuentes de datos**: Ahora puedes conectar Snowflake y Azure Data Lake.
- **Automatización mejorada**: Agregamos la posibilidad de programar tareas recurrentes sin necesidad de intervención manual.

## 🛠️ Mejoras

- Optimización del rendimiento en la carga de datos.
- Mejoras en los logs de ejecución para una mayor trazabilidad.
- Reducción del tiempo de respuesta en consultas complejas.

## 🐞 Correcciones de errores

- Se corrigió un error que impedía la conexión a ciertas bases de datos OData.
- Solucionado un problema con los permisos de usuario en la exportación de datos.
- Mejorada la estabilidad en entornos con alta concurrencia.

## 📌 Notas adicionales

- Esta versión requiere actualización de los permisos de API para habilitar las nuevas funciones.
- Si encuentras algún problema, por favor repórtalo en nuestro [centro de soporte](#).
